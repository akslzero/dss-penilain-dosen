# dss-penilain-dosen
DSS Penilaian Dosen - AHP, SAW, TOPSIS

Aplikasi ini mengimplementasikan metode AHP (Analytical Hierarchy Process) untuk pembobotan kriteria, serta SAW (Simple Additive Weighting) dan TOPSIS (Technique for Order of Preference by Similarity to Ideal Solution) untuk perangkingan 30 data dosen dengan nama akademis
